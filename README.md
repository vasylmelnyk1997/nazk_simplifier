# NAZK_Simplifier / Спрощувач НАЗК декларацій 

Що робить скрипт:
1. прибирає купу сміття (незначущю інформацію, наприклад: "[Конфіденційна інформація]", "[Не застосовується]", etс.
2. об'єднує ПІБ в одне поле
3. формуює адресу в одне поле
4. формуються списки нерухомості (готові до копіюваня в дос'є по кліку)
5. так само формуються списки ТЗ
6. рахується загальна сума доходу (з групуванням по особі)
7. бейдж року з випадаючим меню доступних декларацій
8. рік і ПІБ в тайтл вікна

## Встановлення експериментального розширення в Chrome

### Як встановити локально в режимі розробника:

1. **Клонуйте репозиторій або завантажте архів:**
   ```bash
   git clone https://github.com/vasylmelnyk1997/nazk_simplifier.git
   ```

2. **Відкрийте Chrome та перейдіть до управління розширеннями:**
   - Введіть у адресному рядку: `chrome://extensions/`
   - Або: Меню → Інші інструменти → Розширення

3. **Увімкніть режим розробника:**
   - Натисніть кнопку **"Режим розробника"** у верхньому правому кутку

4. **Завантажте розширення:**
   - Натисніть **"Завантажити розпаковане розширення"**
   - Оберіть папку проекту (там де знаходяться файли розширення)

5. **Готово!** Розширення з'явиться у списку активних розширень

### Розробка та відладка:

- При внесенні змін у код, натисніть кнопку **"Перезавантажити"** ↻ поруч з розширенням
- Для контролю ошибок натисніть **"На помилки ознайомитися"** (якщо вони з'являються)
- Відкрийте DevTools розширення натиснувши на нього для детального дебагу
